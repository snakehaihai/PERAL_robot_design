# PERAL Robot Design

[ICRA 2026] PERAL: perception-aware motion control for passive LiDAR excitation in spherical robots

## Project Overview

This repository contains all the design files, code, and documentation for the PERAL (Perception-Aware Motion Control) spherical robot project. The project focuses on developing advanced motion control algorithms that enable passive LiDAR excitation in spherical robots.

## Getting Started

### For Students Contributing Code:
1. Navigate to the appropriate subdirectory in `student_code/`
2. Read the README for contribution guidelines
3. Implement your code with clear documentation
4. Test thoroughly before committing
5. Push your changes with descriptive commit messages

### For Students Contributing Designs:
1. Navigate to the appropriate subdirectory in `design/`
2. Follow the naming conventions in the README
3. Include both native and universal file formats when possible
4. Document your design decisions
5. Update documentation to match your changes

### For Sourcing Information:
1. Navigate to the appropriate subdirectory in `sourcing/`
2. Use the provided template for listing components
3. Include vendor links and pricing information
4. Check with supervisor before making purchases

## Collaboration Guidelines
