# Mechanical Design

This directory contains mechanical specifications, technical drawings, and design documents.

## Contents

### Drawings (`drawings/`)
Technical drawings including:
- Dimensional drawings
- Assembly views
- Section views
- Detail views
- Tolerance specifications

### Specifications (`specs/`)
Mechanical specifications including:
- Material specifications
- Manufacturing tolerances
- Surface finish requirements
- Assembly requirements
- Testing procedures

## File Formats

- **PDF**: For easy viewing and printing
- **DXF**: For 2D drawings (CAM/laser cutting)
- **DWG**: Native AutoCAD format
- **Engineering drawings**: Include title blocks with part numbers, revisions, etc.

## Drawing Standards

Follow standard engineering drawing practices:
- Use standard views (front, top, side)
- Include dimensions and tolerances
- Specify materials
- Add notes for critical features
- Include scale and units
